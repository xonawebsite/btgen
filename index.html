BTGen

BTGen v0.1.4

The web development boilerplate CLI generator tool.

Introduction

BTGen is a tiny CLI tool to generate a nice starting point for many web development projects, like React, Vue, Express or simple plain HTML.

In order to install the tool you'll need Node.js and npm installed in your machine. If you already have both requirements, let's get started!

Install

To install the tool, all you have to do is type the following command in your terminal / cmd / powershell or whatever you use as command line execution. $ npm i -g btgen

Of course, you can install it using yarn instead, if you like to. $ yarn global add btgen That's it!

Also you can install it locally following these simple steps:

  1. Clone this repository with:
    $ git clone https://github.com/kenliten/btgen.git
  2. Go to the project folder:
    $ cd btgen
  3. Link it to your environment with:
    $ npm link

Another option is run the code without install it in your system:

  1. Clone this repository with:
    $ git clone https://github.com/kenliten/btgen.git
  2. Go to the project folder:
    $ cd btgen
  3. Run your commands directly with nodejs:
    $ node index.js <template> <project-name>

Templates

With BTGen, you have available some templates to start working faster, and spend less time creating the project structure.

To use a template, the command structure is the following:

$ btgen <template-name> <project-name> [options]

If you want to create a simple html + css + javascript/jquery webpage, and the project's name is fancy-site, you'll need this command:

$ btgen webpage fancy-site

BTGen will create the project folder and the content inside, so you don't need to create manually the folder, just run the commands and open your brand new project.

Available Templates

Those are the available templates right now, you can use and test each one separately.

  • webpage
  • bootstrap
  • webgame
  • vuepage
  • reactpage
  • expressapp
  • angularpage
  • electronapp

Each template can take several options to change the result boilerplate, to see the available options use this:
btgen <template-name> --help -v
